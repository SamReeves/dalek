# dalek
Twitter bot with some basic features intended for use by folloers
@unfareNYC

Current features:     
	Tweet lines from a text file.    
	Follow every follower back    

Features in development:
	Build in a Docker container
	Deploy to heroku

	Read DMs
	Respond to DMs with instructions

	keep a list of verified users who can:
		trigger a tweet with a #
		share location
		remove themselves from the list

	keep a list of admins who can:
		add users
		subtract users
		remove themselves from this list
