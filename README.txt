# dalek
Twitter bot with some basic features intended for use by folloers
@unfareNYC

Current features:     
	Tweet lines from a text file.    
	Follow every follower back    
	Read DMs
	Keep list of admins and users
	Admins can add/subtract users
	keep local cache of all content
	Respond to DMs with message

Features in development:
	Deploy to AWS
